================================================================
  PDF Image Upscaler - 빠른 시작 가이드
================================================================


  ● Mac 사용자  →  install_mac.command 더블클릭
  ● Windows 사용자  →  install_windows.bat 더블클릭

  끝입니다. 나머지는 전부 자동입니다.


================================================================
  처음 실행하면 이런 일이 일어납니다
================================================================

  1. 내 컴퓨터 사양을 자동 체크합니다
  2. "설치할까요?" 물으면 → Y 누르세요
  3. Python + AI 모델 자동 설치 (첫 회만, 2~5분)
  4. 브라우저가 열립니다
  5. PDF 끌어다 놓고 → "2x 업스케일 시작" 클릭 → 완료!


================================================================
  알아두면 좋은 것
================================================================

  - 최대 15장, 100MB까지
  - 15장 기준 약 1~5분 (컴퓨터 성능에 따라 다름)
  - 내 컴퓨터에서만 처리됨 (인터넷 업로드 없음)
  - 두 번째부터는 바로 실행됨 (설치 과정 없음)
  - 종료: 터미널 창 닫기 또는 Ctrl+C


================================================================
  안 될 때
================================================================

  Mac "확인되지 않은 개발자" 경고
  → 파일 우클릭 > 열기

  Windows "PC 보호" 경고
  → "추가 정보" 클릭 > "실행"

  브라우저가 안 열릴 때
  → 직접 입력: http://localhost:7860

  그래도 안 되면
  → 사용설명서.txt의 "문제 해결 가이드" 참고
